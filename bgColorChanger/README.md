# Background Color Changer

A simple React + Vite application that allows users to change the background color of the page with a button click.

## Features

- Click a button to generate a random background color
- Clean and minimal UI
- Fast development with Vite and HMR
- Built with React

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

Create a production build:

```bash
npm run build
```

### Preview

Preview the production build:

```bash
npm run preview
```

## How to Use

1. Start the development server
2. Click the "Change Color" button to randomly change the background color
3. The color value is displayed on the screen
