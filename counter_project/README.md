
# Counter Project

A simple React application demonstrating state management using hooks.

## Features

- Increment and decrement counter
- Display current count
- Reset functionality

## Installation

```bash
npm install
```

## Running the Project

```bash
npm start
```

The app will open at `http://localhost:3000`.

## Project Structure

```
counter_project/
├── src/
│   ├── App.js
│   └── index.js
├── public/
├── package.json
└── README.md
```

## Technologies Used

- React
- JavaScript


## Usage

Click the buttons to increment, decrement, or reset the counter value.
